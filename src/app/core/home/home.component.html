Welcome to Kewl Music!

This is an example the awesome sauce that is Angular2!

Feel free to browse artists and albums. Notice the following functionality provided in this demo:

  • Multiple layouts
  • Reusable components: loading, pager, back
  • Directives: Logoff
  • Services: AuthService, ArtistService, AlbumService
  • Built using Modules: Auth, Artist, Album, Core
  • Parent and Child routing
  • Auth guard to prevent unauthorized access to any page
  • Login uses FormGroup to define form elements and validation
  • Web Api is implemented using OData so all data requests are shaped by our service!
  • Paging implemented for all lists
  • Search for all lists
  • Paging respects current search filter
  • Album art is stored as a base64 encoded string in the db. Album shows how to use a property of the model to display the image in an img tag
